5. Рисовалка

  • Для рисования выберите нужный цвет щелчком по нему левой кнопкой мыши, а затем проведите курсором по полю с нажатой левой кнопкой мыши.
  • В зависимости от скорости движения мыши можно получить разные линии.
  • Для стирания щёлкните по первому фрагменту слева, а затем проведите курсором по элементам рисунка с нажатой левой кнопкой мыши.
  • Для полного обновления нажмите клавишу F5.
