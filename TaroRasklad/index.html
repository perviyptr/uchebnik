Расклады на картах Таро

В стандартной колоде Таро 78 карт. Младшие Арканы (56 карт) состоят из четырёх мастей:

  • Жезлы (Посохи);
  • Кубки (Чаши);
  • Мечи;
  • Пентакли (Денарии).
Каждая масть содержит по 14 карт.

Старшие Арканы (22 карты) имеют большое количество взаимосвязанных значений:

Старшие Арканы служат для предсказания событий, имеющих серьёзное влияние на судьбу человека, и служат для долговременных прогнозов. Так, с их помощью можно узнать, каким будет следующий период жизни, какими характеристиками обладает личность, на которую делается расклад карт.

Младшие арканы рекомендуется использовать, чтобы узнать, как будет развиваться какая - либо конкретная тенденция или событие в жизни человека. Они служат для составления кратковременного прогноза по сложившейся жизненной ситуации.

Из числа Младших Арканов можно выбрать карту, которая будет символизировать человека, на которого гадают или который фигурирует в гадании. Каждый Младший Аркан может соответствовать человеку вполне определённого возраста и пола. Так, Рыцарь как правило соответствует молодому мужчине, Король зрелому или старику, Паж символизирует ребёнка или подростка, а Королева женщину. Однако Королева иногда может обозначать и мужчину, а Паж и Рыцарь — женщину.

На странице Трактовка карт представлены все карты колоды Таро с разбивкой по Арканам и мастям. По щелчку левой кнопкой мыши по карте откроется в отдельном окне трактовка выбранной карты.

Выбрав одну или несколько понравившихся карт на этой странице можно узнать бессознательные мотивы, которые управляют вашим поведением.

Получить характеристику своего душевного состояния на текущий момент можно, выбрав случайную карту здесь.

Ниже приведены ссылки на расклады для гадания с помощью карт Таро. Для создания расклада выбирайте закрытые карты левой кнопкой мыши, а затем проанализируйте трактовку карт.
Помните, что трактовки прямой и перевёрнутой карты отличаются.

  • Расклад «Взаимосвязь денег и работы»
  • Расклад «Кельтский крест» (используются все карты колоды)
  • Расклад «Кельтский крест» (используются только Старшие Арканы)
  • Расклад «Корень проблемы»
  • Расклад «Что мешает нам быть вместе?»
  • Расклад «Познай себя»
  • Расклад «Проблемы на работе»
  • Расклад «Проблема в отношениях»
  • Расклад «На день рождения»
