5. Полезные советы для новичков при работе в SolidWorks

    Ориентировать виды можно с помощью следующих сочетаний клавиш:
  • Ctrl + 1 — вид спереди;
  • Ctrl + 2 — вид сзади;
  • Ctrl + 3 — вид справа;
  • Ctrl + 4 — вид слева;
  • Ctrl + 5 — вид сверху;
  • Ctrl + 6 — вид снизу;
  • Ctrl + 7 — изометрия;

  • Для вращения объектов можно использовать клавиши со стрелками;
  • для перемещения объектов можно использовать клавиши со стрелками с нажатой клавишей Ctrl;
  • для вращения объектов по часовой стрелке и обратно — клавиши со стрелками с нажатой клавишей Alt;
  • для уменьшения объекта можно использовать клавишу z, для увеличения Shift + z;
  • открытие меню Ориентация производится клавишей Пробел;
  • для создания дополнительного компонента сборки перетащите его из дерева модели в окно с нажатой клавишей Ctrl;
  • Ctrl + C — копировать эскиз в дереве модели;
  • Ctrl + V — вставить эскиз в выбранную плоскость;
  • можно скруглять непересекающиеся элементы эскиза;
  • для проецирования граней на эскиз используется команда Преобразование объектов;
  • для копирования компонентов, элементов и объектов можно использовать перетаскивание с нажатой клавишей Ctrl;
  • для разворота грани модели или плоскости эскиза параллельно экрану щёлкните по ним или по их названию в дереве модели и выберите значок со стрелкой Перпендикулярно.
