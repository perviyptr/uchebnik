1. Составить картинку из фрагментов

  • Составьте исходную картинку из 16 фрагментов, расположенных ниже.
  • Для вставки фрагмента щёлкните левой кнопкой мыши по выбранному фрагменту, а затем по ячейке клетчатого поля.
  • Для стирания щёлкните по первому пустому фрагменту слева, а затем по нужному фрагменту на поле.
  • Для полного обновления нажмите клавишу F5.
