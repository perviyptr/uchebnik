4. Живая картинка

  • Нарисуйте живую картинку используя элементы, расположенные ниже.
  • Для рисования выберите нужный элемент щелчком по нему левой кнопкой мыши, а затем проведите курсором по ячейкам клетчатого поля с нажатой левой кнопкой мыши.
  • Для вставки фрагмента щёлкните левой кнопкой мыши по выбранному фрагменту, а затем по ячейке клетчатого поля.
  • Для стирания щёлкните по первому пустому фрагменту слева, а затем по нужному фрагменту на поле, либо проведите курсором по всем элементам с нажатой левой кнопкой мыши.
  • Для полного обновления нажмите клавишу F5.
