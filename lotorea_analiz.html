Можно ли выиграть в лоторею? Проверь сам!

С помощью этой простой програмки можно проверить свою удачу на выигрыш в лоторею, не потратив при этом ни копейки.
Для этого укажите количество билетов, которые будут участвовать в тираже(по умолчанию 5) и после выбора типа лотореи
(6 из 45, 5 из 36 или 7 из 49) щёлкните по кнопке "Сгенерировать случайные номера в билетах".

Программа предложит вам добавить свои варианты чисел и после щелчка по "ОК" сгенерирует случайные числа во всех билетах
(в одном варианте разные для всех билетов, а в другом - одинаковые).
Числа в поля раздела "Билеты, заполненные вручную" введите сами через запятую (если не хотите добавлять свои варианты - можно поля оставить пустыми).

После щелчка по кнопке "Сгенерировать результат тиража" выпадут выигрышные цифры, а после щелчка по кнопке
"Определить выигрыш" вы можете увидеть количество угаданных номеров и ваш возможный выигрыш
(но скорее всего это будет проигрыш!).
Если хотите попробовать ещё раз щёлкните по кнопке "Создать новый набор".

После этого можно пойти в магазин и на сэкономленные деньги от покупки лоторейных билетов купить что-нибуть полезное!
